# Relatório Operacional - Dashboard Web

Projeto React + Vite para upload de Excel, análise operacional e geração de relatório PDF A4.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy na Vercel

1. Suba estes arquivos no GitHub.
2. Na Vercel, clique em New Project.
3. Importe o repositório.
4. Framework: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
